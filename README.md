# ReactJS Fullstack website for Russian Tech Platform

## Tool & Stack:
- React
- fetch API
- Flask
- Docker
---
## Frontend setup
```
npm install
```

```
yarn
```

## Run frontend
```
yarn run start
```
```
npm start
```

---
## Docker run fullstack application
```
docker build . -t rutech-reactjs
docker run -p 3000:3000 -d dockerized-react
```
